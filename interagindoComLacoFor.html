Para alterar um elemento no Vue.js é necessário que seja utilizado métodos javascript.
Para adicionarmos um elemento utilizamos o método push.
Para removermos um elemento, utilizamos o método pop.
Para modificarmos um elemento utilizamos o método nativo do vue $set que recebe 3 parâmetros: o data-binding que deve ser alterado, o índice e o conteúdo a ser inserido
